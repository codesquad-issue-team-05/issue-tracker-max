## 🤝🏼 Collaboration

### Branch Convention

Postfix feature branch with issue #.

작업 repository = 선택 사항

작업 repo (PR)-> BE or FE (PR)-> dev (PUSH)-> release

dev-be/feature/#이슈번호
fe/feature/#이슈번호

### Commit Convention

Prefix with Issue #        
Ex: #1 feat: add new feature

> `feat` 기능 구현

> `fix` 버그 수정

> `perf` 성능 개선

> `refactor` 단순 리팩터링

> `test` 테스트 관련

> `build` 빌드 관련

> `ci` ci 관련

> `chore` 그 외 단순 수정

> `docs` 문서 관련

ex) #이슈번호 feat: issue 로그인 구현
